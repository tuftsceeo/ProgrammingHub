* **LV17_ssh** contains a Python script for PySSH and a LabVIEW Interface, written by Chris Rogers

The invoke_shell parameter is used to start an interactive shell session (pseudo-terminal) over an SSH server. More information can be found in the <a href="http://docs.paramiko.org/en/2.5/api/client.html">Paramiko Documentation</a>.
